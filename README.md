# Tubes-Stima: Robocode Tank Royale

Tugas Besar Strategi Algoritma - Implementasi Strategi Greedy dan Heuristik pada Bot Robocode.

## Deskripsi Singkat
Proyek ini berisi implementasi bot untuk kompetisi Robocode Tank Royale. Terdiri dari satu bot utama (MainBot) yang menggunakan strategi Greedy terbaik, serta tiga bot alternatif (AltBot1, AltBot2, AltBot3) dengan variasi strategi yang berbeda.

## Struktur Folder
- `/src`: Berisi kode sumber (C#) untuk MainBot dan ketiga AltBot.
- `/doc`: Berisi laporan tugas besar dalam format PDF.

## Strategi yang Digunakan
1. **MainBot**: Menggunakan strategi [Sebutkan strategi singkat, misal: Greedy Target Terdekat].
2. **AltBot 1-3**: Variasi strategi pergerakan dan penembakan berbasis heuristik.

## Cara Menjalankan
1. Pastikan .NET SDK terbaru sudah terinstal.
2. Buka aplikasi Robocode Tank Royale.
3. Tambahkan folder `src` ke dalam daftar folder bot di aplikasi.
4. Pilih bot dan mulai simulasi.
